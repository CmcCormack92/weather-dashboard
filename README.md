# Weather Dashboard

## Purpose

A basic weather dashboard that will show you the current weather and a 5-day forecast of the input location. It will save your locations to quickly reference in the future.

## Built with

* HTML
* Bootstrap
* JavaScript
* One Call Weather API
* Moment.js
